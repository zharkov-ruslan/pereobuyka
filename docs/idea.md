# Бот-консультант для записи в сервис шиномонтажа

## 1. Название проекта

Бот-консультант для записи в сервис шиномонтажа

## 2. Краткое описание идеи

Это Telegram-бот, с которым можно переписываться как с консультантом сервиса: уточнить услуги, подобрать подходящие варианты и записаться на удобное время. У бота есть актуальный прайс с учётом длительности работ, рабочий календарь и расписание записей. После визита фиксируются оказанные услуги; для постоянных клиентов действует программа лояльности с бонусами, которые можно тратить на следующие визиты.

## 3. Для кого этот бот

Для клиентов сервиса шиномонтажа — в первую очередь для автовладельцев, которым нужна консультация по услугам и запись без лишних звонков. Для самого сервиса бот — способ принимать обращения и записи в одном понятном канале.

## 4. Какую задачу решает

Помогает клиенту быстро понять, что предлагает сервис, сколько это занимает по времени и во сколько есть свободные окна, чтобы записаться без согласования «на словах». Снимает часть нагрузки с администратора по типовым вопросам и записи, даёт прозрачную историю оказанных услуг и стимулирует возвращаться за счёт бонусов.

## 5. Что умеет бот

- Работает как Telegram-бот.
- Регистрирует нового пользователя.
- Ведёт диалог как консультант: помогает выбрать услугу и оформить запись.
- Отвечает на вопросы пользователя в рамках своей роли по услугам сервиса.
- Ведёт рабочий календарь сервиса и график записи клиентов.
- Ведёт прайс-лист услуг с нормированием по времени, чтобы бронировать точные временные слоты.
- Даёт выбрать нужную услугу или несколько услуг и свободное время в графике работы сервиса для записи.
- Ведёт учёт оказанных пользователю услуг.
- Реализует программу лояльности: начисляет бонусы за оказанные услуги и позволяет списывать бонусы в счёт оплаты следующих услуг.

## 6. Примеры вопросов к боту

- Сколько стоит смена комплекта шин и сколько это займёт по времени?
- Есть ли свободное окно завтра после обеда?
- Что входит в шиномонтаж и нужна ли отдельно балансировка?
- Можно записаться на шиномонтаж и развал-схождение в один визит?
- Какие услуги вы делаете для легковых авто?
- Как записаться на смену резины на конкретную дату и время?
- Сколько у меня бонусов и как их можно использовать?
- Какие ближайшие свободные окна для записи на этой неделе?
