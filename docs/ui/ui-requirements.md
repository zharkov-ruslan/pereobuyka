# Требования к интерфейсу

Стиль https://www.tbench.ai/

## Экран 1: Панель преподавателя

Я хочу открыть один экран и сразу понять, как живёт поток — без кликов, без поиска.

**Вверху — цифры по потоку:**
Четыре блока с ключевыми показателями: 
- сколько студентов в потоке
- сколько домашек сдано
- какой средний прогресс по группе
- сколько вопросов задали боту за последние две недели. 
Рядом с каждой цифрой — небольшое сравнение с прошлой неделей, чтобы видеть динамику.

**График активности + лента вопросов:**
Под цифрами — линейный график: сколько вопросов студенты задавали боту каждый день за последние две недели или другие периоды (можно сделать несколько предустановленных периодов). Хочу видеть, когда группа «оживает» — перед дедлайнами, после занятий. 
Рядом с графиком или прямо под ним — лента последних вопросов: кто спросил, когда, текст вопроса и ответ ассистента. Постраничная навигация и поиск по вопросам.

**Последние сдачи:**
Справа — живая лента: кто из студентов что сдал и когда. Имя, урок, время. Если кликнуть — вижу текст отчёта студента и ссылки на его работы.

**Матрица прогресса:**
Внизу — таблица: строки — студенты, столбцы — уроки. В каждой ячейке галочка если сдано, пусто если нет. В последнем столбце счёт — сколько из скольких. Студенты отсортированы по счёту. Если навести на галочку — вижу дату сдачи. Хочу с первого взгляда понимать, кто отстаёт.

## Экран 2: Лидерборд

Это публичный экран — его видят и студенты, и преподаватель. Он должен мотивировать.

Вверху — переключатель между двумя видами: таблица и координатная карта.

**Таблица:**
Список студентов по убыванию прогресса. У каждого — место в рейтинге, прогресс-бар, счёт (например, 6 из 10), иконки по каждому уроку: сдал или нет. Топ-3 выделены золотом, серебром, бронзой.

**Координатная карта:**
Каждый студент — точка на плоскости. По горизонтали — продвижение по урокам, по вертикали — выполнение домашек. Навести на точку — увидеть имя и цифры.

## Экран 3: Личный кабинет студента

Студент заходит и видит свой личный прогресс.

Вверху — приветствие по имени, рядом кнопка перехода в чат с ассистентом.

Ниже — краткий статус: сколько уроков сдано из общего числа и какой следующий урок.

Потом — список всех уроков: сданные отмечены зелёным с датой, текущий — с кнопкой «Сдать задание», будущие — заблокированы.

**Форма сдачи задания:**
По клику «Сдать задание» открывается форма: поле для текстового отчёта (что сделал, что понял) и поле для ссылок на работы. После отправки урок отмечается как сданный.


## Чат с AI-ассистентом (глобальный)

Чат доступен на любом экране — не отдельная страница, а плавающая кнопка в углу. Нажал — открылась панель поверх текущего контента, закрыл — вернулся туда, где был.

Внутри панели: история переписки сверху, поле ввода снизу. Старые вопросы и ответы видны, можно прокрутить вверх. Новый вопрос — пишешь, отправляешь, ответ появляется с небольшой задержкой.

Визуально напоминает терминальный чат или IDE-ассистент: тёмный фон, сообщения пользователя справа, ответы ассистента слева с иконкой.
