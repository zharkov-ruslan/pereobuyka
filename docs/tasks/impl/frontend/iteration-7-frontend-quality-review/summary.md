## Исправления по ревью

1. **`admin-create-booking-sheet.tsx`** — правило `react-hooks/set-state-in-effect`: сброс полей формы при открытии панели больше не делается синхронным `setState` в начале `useEffect`. Логика вынесена во внутренний компонент `AdminCreateBookingSheetBody` с **`key={slot.starts_at}`** (при закрытии панели тело не монтируется): состояние сбрасывается размонтированием, загрузка клиентов/услуг остаётся в эффекте без каскадных синхронных обновлений.

2. **`app/admin/page.tsx`** — закрыт `react-hooks/exhaustive-deps`: `loadDashboard` обёрнут в стабильный `useCallback` ([]), актуальное `data` для выбора недели и флага глобального лоадера читается через **`dataRef`**, обновляемый в отдельном `useEffect` (требование `react-hooks/refs`: не присваивать `ref.current` в теле рендера).
